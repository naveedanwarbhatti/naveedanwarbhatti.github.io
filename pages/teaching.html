Open-source tool for the academic community

Google Scholar Venue Ranker (GSVR) adds CORE conference tiers and SJR journal quartiles directly on Google Scholar profile pages, helping researchers quickly interpret venue prestige while evaluating publication lists.
Install Chrome Extension View on GitHub
  • Shows historical CORE ranks (A*, A, B, C) for conferences/workshops by publication year.
  • Adds SJR quartiles (Q1–Q4) for journals by publication year.
  • Includes a summary panel that aggregates ranks across publications for quick scanning.
  • Uses DBLP-assisted venue matching to disambiguate venue names.
If you use Google Scholar regularly for hiring, promotion, or self-assessment, this extension saves time and reduces manual lookups.
